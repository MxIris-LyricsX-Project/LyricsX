# What's New

## New

- **Opt in to prerelease builds** — A new *Receive beta updates* switch under *Preferences → Updates* (off by default).
- **Apple Music style lyrics panel now works on macOS 12–14** — Previously required macOS 15. The overall minimum supported macOS is now 12; users on macOS 11 will not receive this or later updates.

## Fixes

- **Desktop lyrics no longer flicker about once a second** — Affects Apple Music, Spotify and other scriptable players.
- **Karaoke fill and scrubber stay aligned with playback** — No longer drift or reset on pause/resume.
- **Pause/resume on the system Now Playing source no longer drifts lyrics** — No backward jump on pause; no forward drift on resume.
- **Apple Music style panel reopens at its last size and position** — No longer jumps back to the default frame.
- **Apple Music style panel has a minimum window size** — Can no longer be shrunk into an unreadable strip.
- **Transport buttons now show hover feedback** — Play / previous / next highlight when hovered.
- **Artwork-based lyric matching no longer rejects the correct cover** — A too-strict brightness check has been relaxed.
